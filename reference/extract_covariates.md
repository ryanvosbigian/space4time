# Extract indices and covariates from space4time capture history objects

Extract indices and covariates from space4time capture history objects.
Note that grouping indices are excluded.

## Usage

``` r
extract_covariates(s4t_ch)
```

## Arguments

- s4t_ch:

  a `s4t_ch` object.

## Value

a list of class `cov_s4t_ch` that contains the indices for \$theta\$ and
\$p\$ parameters.
